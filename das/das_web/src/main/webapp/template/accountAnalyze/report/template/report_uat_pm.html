账户诊断阅读指引
主要指标：
胜率：一般情况下评判客户对市场走势判断的准确度，客户对市场的敏感程度。
平均盈亏比：评价客户的操作水平，对盈利、亏损、风险的管控。优秀操作客户会把平均盈亏比保持在一个较高的水平。
最大盈亏比：判断客户的极端亏损单是否对整体交易造成明显影响。
持仓保证金水平走势图：判断客户仓位水平的控制。
持仓时间-盈亏散点图：判断客户是否存在“抗单”的现象。

建议客户对交易中部分典型的交易进行复盘，如盈利最大的某几单交易或亏损最大的某几单 交易，弄明白为何盈利，为何亏损。总结出合适自己的交易方式与交易理论。
